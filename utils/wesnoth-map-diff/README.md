## Setup

1 - Make sure you have Node on your system. If you don't have, install it. You can use [nvm](https://github.com/nvm-sh/nvm).

2 - Install the project dependencies:

```
npm i
```

3 - Build it:

```
npm run build:dev
```

4 - Run it:

```
node ./build/index.js old.map new.map output.png
```

## Contributing

```
npm run test
```
